# Tech Stack — Portfolio

## Frontend

- **Framework:** React 18
- **Language:** JavaScript (JSX) / TypeScript
- **Routing:** React Router v6
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion

## Build & Tooling

- **Bundler:** Vite
- **Package manager:** npm
- **Linting:** ESLint
- **Formatting:** Prettier

## Deployment

- **Hosting:** Vercel / GitHub Pages
